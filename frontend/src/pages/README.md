# Pages Directory

This directory contains key routing views grouped by module domains:
- `auth/` — Login, Forgot Password, Reset Password
- `dashboard/` — Custom metrics and alerts for Admin, Coordinator, Member, and Auditor
- `sessions/` — Scheduled events, scheduling wizards, and timelines
- `members/` — Member rosters and bulk import interfaces
- `attendance/` — Verification interfaces and logs
- `activities/` — Submission inputs and review queues
- `reports/` — Data sheets and export settings
